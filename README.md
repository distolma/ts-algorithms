## About 
This repository contains **TypeScript** implementations of different famous Computer Science algorithms.

Inspired by [javascript-algorithms](https://github.com/mgechev/javascript-algorithms).

## Algorithms

### Done
- Bubble Sort
- Factorial
- Fibonacci
- FizzBuzz
- Insertion Sort
- Merge Sort
- Selection Sort
- Shuffle
- Sieve of Eratosthenes
